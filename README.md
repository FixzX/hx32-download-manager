# hx32 download manager

a modern linux download manager built with python and gtk4.

## install

1. install gtk4 and python gi bindings:

```bash
sudo apt install python3-gi python3-gi-cairo gir1.2-gtk-4.0
```

2. run the app:

```bash
cd /home/f1xz/Téléchargements/Project
python3 hx32_download_manager.py
```

## features

- real file download from url
- dark purple interface
- tabs for downloads, themes, and settings
- change download folder
- cancel download
